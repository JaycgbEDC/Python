## 二、因数、因子有关

1. 概念：
   * 因数（约数）：一个数，如果存在可以被它整除的数，则这些数都是该数的因数。
     * 如4的因数：1，2，4
     * 0没有因数
   * 因子：一个数，如果存在可以被它整除的数且这些数不包括它本身，则这些数都是该数的因子
     * 如4的因子：1，2
   * 两个数或多个数的最大公约数：约数中最大的那个
2. 求一个数的因子或因数
3. 求两个数的最大公约数：
   * 库math.gcd()
   * 辗转相除法：
     * 
     * ```python
       def gcd(a, b):
           while b:
               a, b = b, a % b
           return a

       ```
4. 求两个数的最小公倍数：最小公倍数等于两数的乘积除以最大公约数
